# Hangman

## Description
A simple and fun hangman game developed by using React + Vite, and CSS. You have 10 guesses to figure out the word. If you can't figure it out, you lose. If you do, congratulations, you win!

This project was developed in the Fundamental Web Technologies course at BCIT.

## Technologies Used
- React JS + Vite
- CSS
